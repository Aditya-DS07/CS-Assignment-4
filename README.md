# Number Guessing Game

## Description
A simple console-based number guessing game built with Python. The computer selects a random number between 1 and 100, and the user has to guess it with hints provided for "Too High" or "Too Low".

## Features
- Random number generation
- Input validation
- Attempt counter
- Replay functionality

## How to Run
1. Ensure Python is installed.
2. Run the command: `python game.py`
